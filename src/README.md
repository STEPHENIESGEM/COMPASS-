# src/

This directory contains all source code for the COMPASS- simulator.

## Structure

```
src/
  ai/          -- AI Companion engine and dialogue system
  engine/      -- 3D twin rendering engine (Three.js / Unity WebGL)
  ar/          -- Augmented Reality overlay layer
  game/        -- Gamification logic: XP, skill trees, scenarios
  api/         -- Backend API layer (Node.js)
  diagnostic/  -- RideWire AI Hub integration and OBD-II interface
  ui/          -- User interface components
  utils/       -- Shared utilities and helpers
```
